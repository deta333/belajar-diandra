Soal Tipe 8: Formulir HTML

Isilah bagian yang kosong agar formulir bisa lengkap dan berfungsi.

1. Buatlah formulir yang mengirim data ke halaman "proses.html"



2. Tambahkan kotak teks untuk nama di dalam formulir



3. Tambahkan kotak email di dalam formulir



4. Buat tombol kirim dalam formulir



5. Buat kotak password di dalam formulir



6. Tambahkan area teks untuk pesan (multiline)



7. Buat pilihan jenis kelamin (Laki-laki / Perempuan) menggunakan radio button



8. Buat pilihan hobi menggunakan checkbox (Membaca, Olahraga, Memasak)



9. Buat menu pilihan kota menggunakan select (Jakarta, Bandung, Surabaya)



10. Tambahkan tombol reset di formulir



11. Buat input angka (number)



12. Buat input tanggal (date)



13. Buat input waktu (time)



14. Buat input file untuk upload



15. Buat form dengan metode GET



16. Buat form dengan metode POST



17. Tambahkan label untuk kotak input nama



18. Tambahkan placeholder "Masukkan nama" pada input nama



19. Buat input yang wajib diisi (required)



20. Gabungkan semua elemen di atas menjadi satu formulir lengkap
