Jawaban Tipe 8: Formulir HTML

Berikut adalah jawaban dan penjelasannya.

1.

<form action="proses.html">
</form>

Formulir harus memiliki atribut action untuk menentukan kemana data dikirim.

2.

<input type="text" name="nama">

Type "text" digunakan untuk kotak teks satu baris.

3.

<input type="email" name="email">

Type "email" akan memeriksa format email yang benar.

4.

<button type="submit">Kirim</button>

Tombol kirim memiliki type="submit" agar data terkirim.

5.

<input type="password" name="password">

Password akan disembunyikan saat diketik.

6.

<textarea name="pesan"></textarea>

<textarea> digunakan untuk teks panjang atau multi-baris.

7.

<input type="radio" name="gender" value="Laki-laki"> Laki-laki
<input type="radio" name="gender" value="Perempuan"> Perempuan

Radio button digunakan untuk memilih satu opsi.

8.

<input type="checkbox" name="hobi" value="Membaca"> Membaca
<input type="checkbox" name="hobi" value="Olahraga"> Olahraga
<input type="checkbox" name="hobi" value="Memasak"> Memasak

Checkbox digunakan untuk memilih lebih dari satu opsi.

9.

<select name="kota">
  <option>Jakarta</option>
  <option>Bandung</option>
  <option>Surabaya</option>
</select>

Select digunakan untuk membuat menu drop-down.

10.

<button type="reset">Reset</button>

Reset menghapus semua data yang diisi di form.

11.

<input type="number" name="umur">

Input angka hanya menerima nilai numerik.

12.

<input type="date" name="tanggal">

Input tanggal memunculkan pemilih kalender.

13.

<input type="time" name="waktu">

Input waktu memunculkan pemilih jam/menit.

14.

<input type="file" name="dokumen">

Input file digunakan untuk upload dokumen atau gambar.

15.

<form action="proses.html" method="GET">
</form>

Metode GET menampilkan data di URL.

16.

<form action="proses.html" method="POST">
</form>

Metode POST menyembunyikan data dari URL.

17.

<label for="nama">Nama:</label>
<input type="text" id="nama" name="nama">

Label membantu menghubungkan teks dengan input.

18.

<input type="text" name="nama" placeholder="Masukkan nama">

Placeholder adalah teks petunjuk di dalam input.

19.

<input type="text" name="nama" required>

Required membuat input wajib diisi sebelum form dikirim.

20.

<form action="proses.html" method="POST">
  Nama: <input type="text" name="nama" placeholder="Masukkan nama" required><br>
  Email: <input type="email" name="email" required><br>
  Password: <input type="password" name="password" required><br>
  Pesan: <textarea name="pesan"></textarea><br>
  Jenis Kelamin:
  <input type="radio" name="gender" value="Laki-laki"> Laki-laki
  <input type="radio" name="gender" value="Perempuan"> Perempuan<br>
  Hobi:
  <input type="checkbox" name="hobi" value="Membaca"> Membaca
  <input type="checkbox" name="hobi" value="Olahraga"> Olahraga
  <input type="checkbox" name="hobi" value="Memasak"> Memasak<br>
  Kota:
  <select name="kota">
    <option>Jakarta</option>
    <option>Bandung</option>
    <option>Surabaya</option>
  </select><br>
  Umur: <input type="number" name="umur"><br>
  Tanggal: <input type="date" name="tanggal"><br>
  Waktu: <input type="time" name="waktu"><br>
  Upload: <input type="file" name="dokumen"><br>
  <button type="submit">Kirim</button>
  <button type="reset">Reset</button>
</form>

Form lengkap ini menggabungkan semua elemen yang telah dipelajari.
