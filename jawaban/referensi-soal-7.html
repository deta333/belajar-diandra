Jawaban Tipe 7 - Form dan Input

1.

<form action="proses.html"></form>

Penjelasan: Tag <form> digunakan untuk membuat form, atribut action menentukan halaman tujuan pengiriman data.

2.

<input type="text" name="nama">

Penjelasan: type="text" untuk membuat kotak input teks.

3.

<input type="password" name="sandi">

Penjelasan: type="password" menyembunyikan karakter yang diketik.

4.

<input type="date" name="tanggal_lahir">

Penjelasan: type="date" menampilkan pemilih tanggal.

5.

<input type="email" name="email">

Penjelasan: type="email" digunakan untuk input alamat email.

6.

<button type="submit">Kirim</button>

Penjelasan: Tombol kirim form menggunakan type="submit".

7.

<input type="color" name="warna">

Penjelasan: type="color" memunculkan pemilih warna.

8.

<textarea name="pesan"></textarea>

Penjelasan: <textarea> membuat kotak teks besar.

9.

<input type="file" name="dokumen">

Penjelasan: type="file" untuk memilih file.

10.

<input type="number" name="angka">

Penjelasan: type="number" hanya menerima angka.

11.

<input type="radio" name="gender" value="L"> Laki-laki
<input type="radio" name="gender" value="P"> Perempuan

Penjelasan: type="radio" membuat pilihan tunggal, name harus sama agar hanya satu yang bisa dipilih.

12.

<input type="checkbox" name="hobi" value="Membaca"> Membaca
<input type="checkbox" name="hobi" value="Olahraga"> Olahraga

Penjelasan: type="checkbox" memungkinkan memilih lebih dari satu.

13.

<input type="tel" name="telepon">

Penjelasan: type="tel" digunakan untuk nomor telepon.

14.

<input type="url" name="website">

Penjelasan: type="url" digunakan untuk alamat website.

15.

<select name="kota">
  <option>Jakarta</option>
  <option>Bandung</option>
  <option>Surabaya</option>
</select>

Penjelasan: <select> membuat dropdown pilihan.

16.

<input type="time" name="waktu">

Penjelasan: type="time" untuk memilih jam.

17.

<form>
  <input type="text" name="nama">
  <button type="submit">Kirim</button>
</form>

Penjelasan: Form sederhana dengan input dan tombol kirim.

18.

<button type="reset">Reset</button>

Penjelasan: Tombol reset menghapus semua isi form.

19.

<input type="text" placeholder="Masukkan username">

Penjelasan: Atribut placeholder memberi teks petunjuk.

20.

<form>
  <input type="email" name="email" placeholder="Email">
  <input type="password" name="sandi" placeholder="Password">
  <button type="submit">Login</button>
</form>

Penjelasan: Form login sederhana dengan email, password, dan tombol kirim.
