Jawaban HTML Dasar - Tipe 1

Berikut adalah jawaban dan penjelasan untuk soal Tipe 1.

1. <h1>Halo Dunia</h1>

Penjelasan: Tag <h1> digunakan untuk membuat judul besar.

2. <p>Saya sedang belajar HTML</p>

Penjelasan: Tag <p> digunakan untuk membuat paragraf.

3. <img src="https://example.com/gambar1.jpg">

Penjelasan: Tag <img> digunakan untuk menampilkan gambar, dan atribut src berisi alamat gambar.

4. <a href="https://google.com">Buka Google</a>

Penjelasan: Tag <a> digunakan untuk membuat link, atribut href adalah alamat tujuannya.

5. <h1>[Nama Kamu]</h1>

Penjelasan: Sama seperti nomor 1, tapi isinya nama kamu sendiri.

6. <p>Hari ini cerah sekali</p>

Penjelasan: Paragraf sederhana untuk teks biasa.

7. <img src="https://example.com/kucing.jpg">

Penjelasan: Menampilkan gambar kucing dari internet.

8. <a href="https://youtube.com">Buka YouTube</a>

Penjelasan: Link untuk membuka YouTube.

9. <h1>Selamat Datang</h1>

Penjelasan: Judul besar dengan teks “Selamat Datang”.

10. <p>Saya suka makan pizza</p>

Penjelasan: Paragraf biasa.

11. <img src="https://example.com/anjing.jpg">

Penjelasan: Menampilkan gambar anjing dari internet.

12. <a href="https://facebook.com">Buka Facebook</a>

Penjelasan: Link menuju Facebook.

13. <h1>HTML itu mudah</h1>

Penjelasan: Judul besar untuk motivasi belajar.

14. <p>Saya sedang belajar membuat website</p>

Penjelasan: Paragraf untuk menjelaskan kegiatan belajar.

15. <img src="https://example.com/bunga.jpg">

Penjelasan: Menampilkan gambar bunga.

16. <a href="https://instagram.com">Buka Instagram</a>

Penjelasan: Link menuju Instagram.

17. <h1>Ayo Belajar!</h1>

Penjelasan: Judul besar untuk semangat belajar.

18. <p>Ini adalah pelajaran pertama saya</p>

Penjelasan: Paragraf tentang pelajaran pertama.

19. <img src="https://example.com/pemandangan.jpg">

Penjelasan: Menampilkan gambar pemandangan.

20. <a href="https://wikipedia.org">Buka Wikipedia</a>

Penjelasan: Link menuju Wikipedia.
