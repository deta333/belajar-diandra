Jawaban HTML Tipe 4

Berikut jawaban dan penjelasan untuk latihan form di HTML.

Jawaban 1

<form>
</form>

Penjelasan: Tag <form> digunakan untuk membuat form.



Jawaban 2

<input type="text" name="nama">

Penjelasan: type="text" membuat kotak input teks biasa.



Jawaban 3

<input type="password" name="sandi">

Penjelasan: type="password" membuat input yang menyembunyikan huruf.



Jawaban 4

<textarea name="pesan"></textarea>

Penjelasan: <textarea> membuat kotak teks besar untuk menulis banyak kata.



Jawaban 5

<input type="submit" value="Kirim">

Penjelasan: type="submit" membuat tombol kirim form.



Jawaban 6

<button>Klik Saya</button>

Penjelasan: <button> membuat tombol biasa.



Jawaban 7

<input type="email" name="email">

Penjelasan: type="email" membuat input khusus email.



Jawaban 8

<input type="radio" name="jk" value="Laki-laki"> Laki-laki
<input type="radio" name="jk" value="Perempuan"> Perempuan

Penjelasan: Radio button digunakan untuk memilih satu dari beberapa pilihan.



Jawaban 9

<input type="checkbox" name="setuju"> Setuju dengan syarat

Penjelasan: Checkbox bisa dicentang atau tidak.



Jawaban 10

<select name="makanan">
  <option>Nasi</option>
  <option>Mie</option>
  <option>Roti</option>
</select>

Penjelasan: <select> membuat daftar pilihan (dropdown).



Jawaban 11

<label>Nama: </label>
<input type="text" name="nama">

Penjelasan: <label> memberi nama di depan input.



Jawaban 12

Nama Depan: <input type="text" name="nama_depan">
Nama Belakang: <input type="text" name="nama_belakang">

Penjelasan: Bisa membuat dua input dalam satu form.



Jawaban 13

<input type="date" name="tgl_lahir">

Penjelasan: type="date" membuat pilihan tanggal.



Jawaban 14

<input type="number" name="umur">

Penjelasan: type="number" untuk angka.



Jawaban 15

<input type="file" name="berkas">

Penjelasan: type="file" untuk upload file.



Jawaban 16

<input type="color" name="warna">

Penjelasan: type="color" membuat pemilih warna.



Jawaban 17

<input type="time" name="waktu">

Penjelasan: type="time" membuat pemilih waktu.



Jawaban 18

<input type="reset" value="Hapus">

Penjelasan: type="reset" menghapus isi form.



Jawaban 19

<form action="proses.html">
</form>

Penjelasan: action menentukan halaman tujuan form.



Jawaban 20

<form>
  Nama: <input type="text" name="nama"><br>
  Email: <input type="email" name="email"><br>
  Pesan: <textarea name="pesan"></textarea><br>
  <input type="submit" value="Kirim">
</form>

Penjelasan: Form lengkap biasanya berisi beberapa input dan tombol kirim.
